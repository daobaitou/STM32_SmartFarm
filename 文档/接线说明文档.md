## 传感器模块

### DHT22 空气温湿度 (AM2302)

| DHT22引脚 | STM32引脚 | 说明 |
|-----------|----------|------|
| VCC (+) | 5V (CH340) | 3.3~5.5V |
| DATA | PA1 | 单总线，模块自带上拉电阻 |
| GND (-) | GND | 共地 |

> 采集周期：2秒

### DS18B20 土壤温度（防水探头）

| DS18B20引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 3.3V | 3.0~5.5V |
| DATA (黄) | PA6 | 单总线，模块自带上拉电阻 |
| GND (黑) | GND | 共地 |

> 采集周期：2秒，12位精度转换需750ms

### FC28 土壤湿度

| FC28引脚 | STM32引脚 | 说明 |
|---------|----------|------|
| VCC | 3.3V | 3.3~5V |
| AOUT | PA0 (ADC1_IN0) | 模拟量输出 |
| GND | GND | 共地 |

> 采集周期：1秒，ADC值范围0~4095

### BH1750 光照强度

| BH1750引脚 | STM32引脚 | 说明 |
|------------|----------|------|
| VCC | 3.3V | 3.3~5V |
| SDA | PB7 (I2C1_SDA) | I2C数据 |
| SCL | PB6 (I2C1_SCL) | I2C时钟 |
| ADDR | GND/悬空 | 地址选择(0x23) |
| GND | GND | 共地 |

> 采集周期：1秒，I2C地址：0x23(ADDR=GND) / 0x5C(ADDR=HIGH)

### MH-Z19B CO2浓度

| MH-Z19B引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 5V | 4.5~5.5V |
| TX | PA3 (USART2_RX) | 交叉连接 |
| RX | PA2 (USART2_TX) | 交叉连接 |
| GND (黑) | GND | 共地 |

> 采集周期：5秒，UART 9600bps 8N1

### BMP180 大气压强

| BMP180引脚 | STM32引脚 | 说明 |
|------------|----------|------|
| VCC | 3.3V | 3.3~5V |
| SDA | PB7 (I2C1_SDA) | 与BH1750共用I2C总线 |
| SCL | PB6 (I2C1_SCL) | 与BH1750共用I2C总线 |
| GND | GND | 共地 |

> 采集周期：2秒，I2C地址：0x77

### LM393 雨滴检测

| LM393引脚 | STM32引脚 | 说明 |
|-----------|----------|------|
| VCC | 3.3V | 3.3~5V |
| AOUT | PA4 (ADC) | 模拟量(备用，可只接DO) |
| DOUT | PA5 (GPIO_EXTI) | 数字输出，有雨=低电平 |
| GND | GND | 共地 |

> 实时检测，中断触发

## 数据存储

### W25Q64 SPI Flash (8MB)

| W25Q引脚 | STM32引脚 | 说明 |
|----------|----------|------|
| VCC | 3.3V | 3.3V供电 |
| GND | GND | 共地 |
| CS | PA15 | 片选（低电平选中），NOJTAG释放 |
| CLK | PB3 | SPI时钟，NOJTAG释放 |
| DI | PB4 | 数据输入(MOSI)，NOJTAG释放 |
| DO | PB5 | 数据输出(MISO) |
| HOLD | 3.3V | 悬空或接高，不使用暂停功能 |
| WP | 3.3V | 悬空或接高，不使用写保护 |

> 软件SPI (Mode 0)，已验证：Manuf=0xEF(Winbond), ID=0x4017(W25Q64)
> 存储用途：系统配置参数(Sector 0) + 传感器数据日志(Sector 1-2)

## 灌溉系统

### YF-S201 水流传感器

| YF-S201引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 5V | 5~18V |
| SIGNAL (黄) | PB0 (EXTI0) | 脉冲输出，下降沿中断计数 |
| GND (黑) | GND | 共地 |

> 实时计数，脉冲频率 ∝ 水流量 (Flow[L/min] = F[Hz] / 7.5)
> 测试方法：吹气模拟，或杜邦线反复触碰GND模拟脉冲

## 电源系统

### 双电源架构

```
12V适配器 ──→ LM2596降压(5V) ──→ STM32(3.3V LDO) + 传感器(5V)
                                        │
18650×2 (7.4V) ──→ 二极管OR电路 ────────┘
                   (市电优先)
```

| 电源 | 输出 | 说明 |
|------|------|------|
| 12V适配器 | 12V | 主电源，经降压后供电 |
| 18650×2 | 7.4V | 备用电池 |
| LM2596 | 5V | 降压模块 |
| STM32 LDO | 3.3V | 板载稳压 |

### 断电检测

| 检测引脚 | STM32引脚 | 说明 |
|---------|----------|------|
| 市电检测 | PA2 (GPIO_EXTI) | 市电断电→下降沿中断 |

## I2C总线共享

```
PB6 (SCL) ─┬── BH1750 (0x23)
            ├── BMP180 (0x77)
            ├── OLED SSD1306 (0x3C)
            └── DS3231 (0x68)
PB7 (SDA) ──┘
```

> 注意：多个I2C设备共用总线，地址不能冲突

---

## SPI总线

```
PB3 (CLK) ─── W25Q64 (8MB Flash)
PB4 (DI)  ─── W25Q 数据输入 (MCU→Flash)
PB5 (DO)  ─── W25Q 数据输出 (Flash→MCU)
PA15 (CS) ─── W25Q 片选
```

> 软件SPI (Mode 0)，关中断保护时序，~1MHz时钟速率
