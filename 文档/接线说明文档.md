## 调试与串口

### DAP调试器 (CMSIS-DAP)

| DAP引脚 | STM32引脚 | 说明 |
|---------|----------|------|
| SWDIO | SWDIO | 数据 |
| SWCLK | SWCLK | 时钟 |
| GND | GND | 共地 |
| 3.3V | 3.3V | 供电(可选) |

### CH340 USB转串口 (printf调试)

| CH340引脚 | STM32引脚 | 说明 |
|-----------|----------|------|
| TXD | PA10 (USART1_RX) | TX接RX，交叉连接 |
| RXD | PA9 (USART1_TX) | RX接TX，交叉连接 |
| 5V | - | 可给外部5V传感器供电 |
| GND | GND | 共地 |

> 串口参数：115200bps, 8N1

## 传感器模块

### DHT22 空气温湿度 (AM2302)

| DHT22引脚 | STM32引脚 | 说明 |
|-----------|----------|------|
| VCC (+) | 5V (CH340) | 3.3~5.5V |
| DATA | PA1 | 单总线，模块自带上拉电阻 |
| GND (-) | GND | 共地 |

> 采集周期：2秒

### DS18B20 土壤温度（防水探头）

| DS18B20引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 3.3V | 3.0~5.5V |
| DATA (黄) | PA6 | 单总线，模块自带上拉电阻 |
| GND (黑) | GND | 共地 |

> 采集周期：2秒，12位精度转换需750ms

### FC28 土壤湿度

| FC28引脚 | STM32引脚 | 说明 |
|---------|----------|------|
| VCC | 3.3V | 3.3~5V |
| AOUT | PA0 (ADC1_CH0) | 模拟量输出 |
| GND | GND | 共地 |

> 采集周期：1秒

### BH1750 光照强度

| BH1750引脚 | STM32引脚 | 说明 |
|------------|----------|------|
| VCC | 3.3V | 3.3~5V |
| SDA | PB7 (I2C1_SDA) | I2C数据 |
| SCL | PB6 (I2C1_SCL) | I2C时钟 |
| ADDR | GND/悬空 | 地址选择(0x23) |
| GND | GND | 共地 |

> 采集周期：1秒，I2C地址：0x23(ADDR=GND) / 0x5C(ADDR=HIGH)

### MH-Z19B CO2浓度

| MH-Z19B引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 5V | 4.5~5.5V |
| TX | PA3 (USART2_RX) | 交叉连接 |
| RX | PA2 (USART2_TX) | 交叉连接 |
| GND (黑) | GND | 共地 |

> 采集周期：5秒，UART 9600bps 8N1

### BMP180 大气压强

| BMP180引脚 | STM32引脚 | 说明 |
|------------|----------|------|
| VCC | 3.3V | 3.3~5V |
| SDA | PB7 (I2C1_SDA) | 与BH1750共用I2C总线 |
| SCL | PB6 (I2C1_SCL) | 与BH1750共用I2C总线 |
| GND | GND | 共地 |

> 采集周期：2秒，I2C地址：0x77

### LM393 雨滴检测

| LM393引脚 | STM32引脚 | 说明 |
|-----------|----------|------|
| VCC | 3.3V | 3.3~5V |
| AOUT | PA1 (ADC) | 模拟量(备用，可只接DO) |
| DOUT | PB0 (GPIO_EXTI) | 数字输出，有雨=低电平 |
| GND | GND | 共地 |

> 实时检测，中断触发

## 灌溉系统

### YF-S201 水流传感器

| YF-S201引脚 | STM32引脚 | 说明 |
|-------------|----------|------|
| VCC (红) | 5V | 5~18V |
| SIGNAL (黄) | PA15 (TIM2_CH1_ETR) | 脉冲输出，外部中断计数 |
| GND (黑) | GND | 共地 |

## 电源系统

### 双电源架构

```
12V适配器 ──→ LM2596降压(5V) ──→ STM32(3.3V LDO) + 传感器(5V)
                                        │
18650×2 (7.4V) ──→ 二极管OR电路 ────────┘
                   (市电优先)
```

| 电源 | 输出 | 说明 |
|------|------|------|
| 12V适配器 | 12V | 主电源，经降压后供电 |
| 18650×2 | 7.4V | 备用电池 |
| LM2596 | 5V | 降压模块 |
| STM32 LDO | 3.3V | 板载稳压 |

### 断电检测

| 检测引脚 | STM32引脚 | 说明 |
|---------|----------|------|
| 市电检测 | PA0 (GPIO_EXTI) | 市电断电→下降沿中断 |

## I2C总线共享

```
PB6 (SCL) ─┬── BH1750 (0x23)
            ├── BMP180 (0x77)
            ├── OLED SSD1306 (0x3C)
            └── DS3231 (0x68)
PB7 (SDA) ──┘
```

> 注意：多个I2C设备共用总线，地址不能冲突
